Formulaire HTML

L'élément HTML <form> représente une section d'un document qui contient des contrôles interactifs permettant à un utilisateur d'envoyer des données à un serveur web.

Un exemple :

Le code derrière tout ça :

<form action="/register" method="POST">
	<label for="name">Nom :</label>
	<input id="name" type="text" name="name">
	<input type="submit" value="Enregistrer">
</form>

Un peu d'explication :

L'attribut method définit la méthode HTTP qui sera utilisée pour envoyer les données au serveur.

Si c'est GET les données du formulaire sont ajoutées à l'URI de l'attribut action avec '?' comme séparateur.

Si c'est POST les données du formulaire sont incluses dans le corps du formulaire et envoyées vers le serveur.

L'attribut action indique le chemin vers le fichier qui traitera les infos transmises via formulaire.
